## 一、设计理念

**Industrial Intelligence OS · 工业大脑 + 霓虹绿 + 赛博朋克**

- 纯黑背景（#000000）→ 极致沉浸，OLED 省电
- 霓虹绿（#00FF00）作为主强调色 → 工业控制感、高对比度
- 毛玻璃卡片 → 层次分明
- 横向 Pipeline 代替竖向时间轴 → 流程一目了然
- 工业 OS 气质 → 不只是 AI 助手，是工业级管控系统

## 二、颜色体系（霓虹绿工业 OS）

### 背景

| 层级 | 色值 | 用途 |
|------|------|------|
| bgRoot | `#000000` | 纯黑背景（OLED 友好） |
| bgSurface | `#0a0a0a` | 次层背景 |
| bgCard | `rgba(20,20,20,0.85)` | 毛玻璃卡片 |
| bgElevated | `#111111` | 抬升元素 |
| bgOverlay | `rgba(0,0,0,0.92)` | 弹窗/遮罩 |

### 边框

| 类型 | 色值 | 用途 |
|------|------|------|
| borderSubtle | `rgba(0,255,0,0.10)` | 卡片默认边框（淡绿） |

## 四、Pipeline（执行链路）— 横向四阶段

这是最核心的 UI 组件，替代原来的竖向时间轴。

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  ● 1     │ → │  ● 2     │ → │  ● 3     │ → │  ● 4     │
│ 需求理解  │    │ 设计系统  │    │ 界面生成  │    │ 工程落地  │
│          │    │          │    │          │    │          │
│ 用户目标  │    │ 色彩字体  │    │ 全套出图  │    │ 组件动效  │
│  与边界  │    │ 间距规范  │    │          │    │          │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

**样式规则：**
- 当前阶段：霓虹绿边框 + 绿色光晕 dot + 数字高亮
- 已完成阶段：绿色边框 + 绿色数字
- 待执行：灰色边框 + 灰色数字
- 各卡片之间有箭头连接线（绿色虚线或箭头）

## 八、禁止事项

| 禁止 | 原因 |
|------|------|
| 纯黑背景换成深蓝灰 | 设计语言完全不同 |
| primary 不用霓虹绿 | 核心品牌色 |
| Pipeline 改回竖向 | 用户定稿是横向四阶段 |
| 文字色用灰色而非白色 | 对比度不足 |
| 卡片无光晕效果 | 失去工业 OS 质感 |
