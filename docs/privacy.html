隐私政策

最近更新：2026 年 5 月 7 日

App Store：AI协作平台 · Bundle ID：com.openclaw.aibrainim

AI协作平台（"我们"、"本应用"或"AI Brain IM"）非常重视您的隐私。本隐私政策说明了我们如何收集、使用和存储您的信息，以及您享有的相关权利。

一、我们收集什么信息

1.1 您主动提交的内容

  • 消息内容：您在对话中输入的文本，以及附加的文件（图片、视频、PDF、代码等）。
  • 上传文件：通过附件入口上传的所有文件，均在传输前经过处理和分片。
  • 指令与任务描述：您在任务面板中创建的指令、优先级设定和任务状态更新。

1.2 自动采集的运行时数据

  • 会话状态：Agent 在线状态、任务执行进度、调度链路状态，用于实时 AI 协作驾驶舱展示。
  • 上传进度：附件的分片上传进度、断点续传状态。
  • 本地存储：对话历史（AsyncStorage）、Gateway 配置偏好。

我们不收集的内容：设备标识符（IDFA）、通讯录、精确位置、麦克风（除您主动拍照/录像）、相册（除您主动选择）等敏感权限未经您操作不会启用。

二、如何使用您的信息

  • 将您的消息路由至 OpenClaw AI 平台，生成 AI 回复并分派至对应 Agent 执行
  • 处理和分析您上传的附件内容（图片识别、文档摘要、代码审查等）
  • 在多 Agent 之间协调任务调度，追踪执行状态并实时回流至本应用
  • 维护跨会话的对话历史和任务上下文
  • 将知识条目收录至您的飞书知识库（仅在您主动触发"知识收录"时）

三、数据存储与安全

3.1 本地存储

  • 对话历史：存储于设备本地（AsyncStorage），可随时在应用内手动清除。
  • 上传队列状态：存储于内存，不持久化，重启后重置。
  • Gateway 配置：您手动填写的连接配置存储于本地，可重置。

3.2 OpenClaw 平台处理

您的消息和上传文件通过 OpenClaw AI 平台处理，具体数据处理说明请参阅：

https://github.com/openclaw/openclaw

3.3 数据保留

  • 本地对话历史：保留至您手动清除或卸载应用
  • 任务记录：最近 48 小时内的活跃任务保留在运行时状态中
  • 知识库收录：写入您指定的飞书知识库，保留策略遵循飞书平台规则

四、数据共享

我们不会将您的个人信息出售给任何第三方。您的数据仅在以下情形下共享：

  • OpenClaw AI 平台：您的消息和上传文件会被发送至 OpenClaw Gateway 进行 AI 处理。
  • 飞书平台：当您触发"知识收录"时，内容会写入您的飞书知识库。
  • 法律要求：如法律或监管要求披露，我们仅在必要的最小范围内提供。

五、您的权利

5.1 清除本地数据

  • 在对话界面点击"清空记录"可清除所有本地对话历史
  • 卸载应用会自动清除所有本地存储数据
  • 在"我的 → Gateway 连接配置"中可重置所有本地配置

5.2 撤回知识库授权

如需撤回知识库写入授权，请在飞书平台移除本应用的访问权限。

5.3 联系我们

如对本隐私政策有任何疑问，或希望行使您的数据权利，请通过以下方式联系我们：

  • GitHub Issues：https://github.com/explore3363-coder/AIBrainIM/issues
  • 应用内反馈：可通过"我的 → 反馈"提交

六、OpenClaw 平台说明

AI协作平台基于 OpenClaw 开源项目构建。OpenClaw 是运行在您本地的 AI Agent 运行时环境，您的消息和文件在本地处理后通过 Gateway 与 AI 模型交互，不经过第三方 AI 服务商的服务器。

七、隐私政策更新

我们可能会不时更新本隐私政策。重大变更将在应用内通过更新说明通知您。最新版本始终发布于：

https://github.com/explore3363-coder/AIBrainIM/blob/main/docs/privacy.html
